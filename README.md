## Contexto do Projeto: 
Você foi contratado(a) por uma escola ou pequeno negócio
para desenvolver um mini-site institucional simples, moderno e responsivo. O
cliente precisa de presença online rápida, layout organizado, botão de contato e
publicação na internet

## O projeto será desenvolvido em 3 etapas:
1) Construção das páginas (HTML)
2) Estilização (CSS)
3) Versionamento com Git e Deploy no Vercel
